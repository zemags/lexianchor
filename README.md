# LexiAnchor — Greek Trainer v1.8

Мобильное PWA-приложение для греческих слов. Сборники, интервальное повторение, ручной ввод, тесты, свайп-пары, изображения-якоря, CSV и переносимая SQLite-база. Аккаунт и сервер для данных не нужны: карточки, изображения и прогресс остаются на устройстве.

## Запуск

### macOS

1. Распакуйте архив.
2. Запустите `start.command`. При блокировке: правой кнопкой → **Open**.
3. Откройте `http://localhost:8080` в Chrome.

### Windows

1. Распакуйте архив.
2. Запустите `start.bat`.
3. Откройте `http://localhost:8080` в Chrome.

Ручной запуск:

```bash
python3 server.py
```

Не открывайте `index.html` через `file://`: WebAssembly и PWA требуют веб-сервер.

## Новое в v1.8

### Ручной ввод ответа

Режим **«Написать ответ»** работает в направлениях греческий → русский и русский → греческий. Можно выбрать строгую проверку, игнорирование греческих ударений или допуск небольших опечаток. После ответа показываются правильный вариант, пример, перевод и подсказка.

### Ежедневная и быстрая тренировка

На главном экране появились:

- **Учить на сегодня** — подошедшие повторения, сложные и новые слова;
- сессии на 2 и 5 минут;
- **Сложные слова** — карточки с повторными ошибками.

Количество новых слов и размер ежедневной сессии задаются в настройках.

### Умные ложные пары

Свайп-режим выбирает ложные переводы по тегам, теме, сборнику и сходству ответа. Например, глагол чаще подменяется другим глаголом, а погодное слово — другим словом о погоде.

### Теги

У карточки есть поле `Теги`. В тренировке можно указать фильтр по тегу. Теги также используются при выборе сложных ложных вариантов.

CSV поддерживает необязательную колонку `Теги`.

### Поиск дублей при импорте

Перед импортом показывается число совпадающих слов. Доступны три действия:

- пропустить дубли;
- обновить заполненные поля существующих карточек;
- создать копии.

Сравнение не зависит от регистра и греческих ударений.

### Изображения-якоря

- поиск Pexels по слову или переводу;
- **поиск по контексту примера**: перевод примера на английский и поиск по полученной фразе;
- основной и дополнительный визуальный якорь;
- режим `contain` или `cover`;
- подпись поверх основного изображения;
- поворот основного изображения на 90°;
- картинки сохраняются внутри SQLite как WebP/BLOB.

Для Pexels вставьте API key: **⚙ → Интеграции → Pexels API key**. Ключ хранится в `localStorage` и не попадает в SQLite или GitHub.

### Статистика и резервные копии

Главная страница показывает точность по режимам, количество сложных слов и повторения на ближайшие семь дней. Приложение напоминает экспортировать `.sqlite`, если база менялась после последней резервной копии.

## CSV

Обязательна только колонка `Слово / фраза`. Поддерживаются табуляция, `;` и `,`.

Ожидаемые колонки:

- Слово / фраза
- Транскрипция слова
- Перевод слова
- Пример на греческом
- Транскрипция
- Перевод примера
- Подсказка / нюанс
- Теги
- Поиск картинки

## Перенос между устройствами

1. Откройте **Импорт и база**.
2. Нажмите **Скачать текущую .sqlite**.
3. Передайте файл на другое устройство.
4. В LexiAnchor выберите **Загрузить .sqlite с устройства**.

В одном файле находятся сборники, карточки, оба изображения-якоря, теги, история ответов и интервалы.

## GitHub Pages

Загрузите содержимое папки в корень репозитория и включите GitHub Pages. После обновления откройте обычный адрес сайта. Кэш версии называется `lexianchor-v1.8.0` и заменяет старый кэш автоматически.

## Техническая основа

- HTML/CSS/JavaScript без фреймворка;
- `sql.js` / SQLite WebAssembly;
- IndexedDB для автосохранения SQLite;
- Web App Manifest и Service Worker;
- Pexels API;
- Chrome Translator → MyMemory → исходный русский запрос.

Облачная синхронизация не включена: перенос между устройствами выполняется через экспорт SQLite, чтобы данные не требовали аккаунта или внешнего сервера.
