# LexiAnchor — Greek Trainer v1.5

Мобильное PWA-приложение для изучения греческого: сборники, интервальное повторение, тест с четырьмя вариантами, изображения-якоря, CSV-импорт и переносимая SQLite-база.

## Что нового в v1.5

- Автоматический перевод русских запросов перед поиском изображений в Pexels.
- Каскад перевода:
  1. встроенный Chrome Translator на компьютере;
  2. бесплатный MyMemory API, если Chrome Translator недоступен;
  3. исходный русский запрос, если оба способа перевода завершились ошибкой.
- Успешный английский перевод сохраняется внутри карточки в SQLite и повторно не запрашивается.
- В окне Pexels показываются исходный русский и фактический английский запрос, а также использованный переводчик.
- Автоперевод работает и в редакторе одной карточки, и в массовом автоподборе изображений.
- В настройках можно отключить автоперевод и указать необязательный email для увеличения бесплатного лимита MyMemory.
- Настройки MyMemory и Pexels API key хранятся только в localStorage конкретного браузера.
- Все возможности v1.3 сохранены: светлая обратная сторона карточки, встроенная галерея Pexels, массовый подбор, тест с четырьмя вариантами и переносимая SQLite-база.

## Куда вставить Pexels API key

1. Откройте LexiAnchor.
2. Нажмите кнопку **⚙ Настройки** в верхней части приложения.
3. Откройте блок **Pexels**.
4. Вставьте ключ в поле **Pexels API key**.
5. Нажмите **Проверить ключ**.
6. Нажмите **Сохранить настройки**.

Ключ сохраняется в `localStorage` текущего браузера. На другом телефоне или компьютере его нужно ввести снова. Он намеренно не включается в экспортируемую SQLite-базу.

## Автоперевод запросов для картинок

Автоперевод включён по умолчанию. Настройка находится в **⚙ Настройки → Автоперевод для поиска**.

Порядок работы:

1. Если запрос уже на английском, приложение сразу передаёт его в Pexels.
2. Если запрос содержит русский текст, приложение проверяет сохранённый английский перевод в SQLite.
3. На поддерживаемом настольном Chrome используется встроенный Translator API. Языковая модель может загрузиться при первом использовании.
4. Если локальный переводчик недоступен или завершился ошибкой, выполняется запрос к MyMemory.
5. Если MyMemory тоже недоступен, Pexels получает исходный русский запрос.

В настройках можно указать email для MyMemory. Это необязательно; email хранится только в `localStorage` текущего устройства и не записывается в SQLite.

После успешного перевода в карточке сохраняются:

- исходный поисковый запрос;
- английский поисковый запрос;
- название использованного переводчика.

Эти данные переносятся вместе с экспортированным `.sqlite`, поэтому на другом устройстве повторный перевод обычно не требуется.

## Поиск и выбор изображения

В редакторе карточки или в **Сборники → Картинки**:

1. Заполните поле **Запрос для картинки** на русском или английском. Русский запрос будет автоматически переведён перед поиском.
2. Нажмите **Найти в Pexels**.
3. Выберите одну из фотографий в модальном окне.
4. Фотография будет загружена, уменьшена до 900×700, преобразована в WebP и сохранена внутри SQLite.
5. Сохраните карточку или нажмите **Сохранить и далее** в Лаборатории картинок.

Кнопка **Использовать первое** выбирает первый результат текущего поиска.

### Массовый автоподбор

В Лаборатории картинок нажмите **⚡ Автоподбор оставшихся**. Для каждой оставшейся карточки приложение использует:

1. значение колонки `Поиск картинки`;
2. перевод слова;
3. исходное греческое слово.

Русские запросы автоматически переводятся по каскаду Chrome Translator → MyMemory → русский fallback. Будет выбран первый результат Pexels. За один запуск обрабатывается максимум 100 карточек, чтобы не расходовать слишком много API-запросов. Автоматически выбранные изображения можно позже заменить вручную.

## Быстрый запуск на ноутбуке

### macOS

1. Распакуйте архив.
2. Дважды нажмите `start.command`. Если macOS блокирует запуск: правой кнопкой → **Open**.
3. Откройте `http://localhost:8080` в Google Chrome.

### Windows

1. Распакуйте архив.
2. Запустите `start.bat`.
3. Откройте `http://localhost:8080` в Google Chrome.

Ручной запуск:

```bash
python3 server.py
```

Не открывайте `index.html` через `file://`: WebAssembly, Service Worker и PWA требуют веб-сервер.

## Установка на телефон

Разместите содержимое папки на GitHub Pages или другом статическом HTTPS-хостинге.

- Android Chrome: меню `⋮` → **Install app / Добавить на главный экран**.
- iPhone Safari: **Поделиться** → **На экран «Домой»**.

Для встроенного поиска Pexels требуется интернет. Уже сохранённые в SQLite изображения доступны офлайн.

## Перенос базы

1. Откройте **Импорт и база**.
2. Нажмите **Скачать текущую .sqlite**.
3. Передайте файл на другое устройство.
4. На другом устройстве выберите **Загрузить .sqlite с устройства**.

Карточки, картинки, прогресс и результаты тестов хранятся в одном SQLite-файле. Pexels API key не переносится и вводится отдельно на каждом устройстве.

## CSV

Обязательна только колонка `Слово / фраза`. Поддерживаются табуляция, `;` и `,`.

Колонки:

- Слово / фраза
- Транскрипция слова
- Перевод слова
- Пример на греческом
- Транскрипция
- Перевод примера
- Подсказка / нюанс
- Поиск картинки — необязательно

## Горячие клавиши

### Карточки

- `Space` — перевернуть карточку
- `1` — Снова
- `2` — Трудно
- `3` — Хорошо
- `4` — Легко

### Тест

- `1`–`4` — выбрать вариант
- `Space` или `Enter` — следующий вопрос

## Техническая основа

- HTML/CSS/JavaScript без фреймворка
- `sql.js` / SQLite WebAssembly
- IndexedDB для автоматического хранения SQLite-файла
- Web App Manifest + Service Worker
- Pexels API для встроенного поиска фотографий
- Chrome Translator API на поддерживаемых настольных версиях Chrome
- MyMemory REST API как бесплатный сетевой fallback

## Изменения v1.5

- Мобильная карточка стала ниже и адаптируется к высоте экрана.
- Кнопки «Снова», «Трудно», «Хорошо», «Легко» размещаются в одну строку и остаются над нижней навигацией.
- Оценка «Снова» больше не вставляет карточку через 1–3 позиции очереди. Карточка возвращается в текущей длинной сессии только после фактического наступления `due_at`; если сессия закончилась раньше, она появится в следующей тренировке после назначенного времени.
